## <a name="now-available-windows-7-year-2-esu-for-csp-partners"></a><a name="6"></a>Windows 7-ESU (2. Jahr) für CSP-Partner jetzt verfügbar

### <a name="categories"></a>Kategorien

- Datum: 29.06.2020
- Typ: Angebote/Märkte
- Status: Jetzt verfügbar

### <a name="summary"></a>Zusammenfassung

Sie finden die Preise für erweiterte Sicherheitsupdates (ESU) für Windows 7 (2. Jahr) ab sofort im [Ressourcenkatalog](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="impacted-audience"></a>Zielgruppe

Alle Partner im CSP-Programm

### <a name="details"></a>Details

Die Unterstützung für Windows 7 wurde am 14. Januar 2020 eingestellt. Während viele Kunden den Umstieg auf Windows 10 bereits eingeleitet haben, ist sich Microsoft bewusst, dass jeder an einem anderen Punkt im Upgradeprozess steht. Die Gültigkeit des Windows 7-ESU (1. Jahr) endet am 12. Januar 2021. Wir wissen, dass Kunden mit der Planung für das nächste Jahr beginnen.  

Für Kunden, die Windows 7-Geräte im Jahr 2021 weiterverwenden möchten, stellen wir CSP-Partnern jetzt Preise für das Windows 7-ESU (2. Jahr) zur Verfügung, damit sie Kunden bei ihrer Entscheidung für Windows 10 unterstützen können. Weitere Informationen finden Sie im [Ressourcenkatalog](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="next-steps"></a>Nächste Schritte
Bitte geben Sie diese Informationen an die entsprechenden Kontakte in Ihrer Organisation weiter.

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>Aktualisierte Funktion zum Entfernen von Beziehungen im Partner Center

### <a name="categories"></a>Kategorien

- Datum: 29.06.2020
- Typ: Funktionen
- Status: Bald verfügbar

### <a name="summary"></a>Zusammenfassung

Ab dem 29. Juni 2020 können CSP Indirect Provider und CSP Indirect Reseller Beziehungen auf untergeordneter Ebene über das Partner Center entfernen und müssen dazu nicht länger ein Partner Center-Supportticket einreichen. So können Partner bei diesen Aktivitäten viel Zeit einsparen.

### <a name="impacted-audience"></a>Zielgruppe

Indirekte CSP-Anbieter (Cloud Solution Provider) und indirekte Handelspartner

### <a name="scenarios-supported"></a>Unterstützte Szenarien:

- Indirekte Anbieter, die über eine Beziehung zu einem indirekten Handelspartner verfügen, können diese Beziehung über das Partner Center entfernen.
- Indirekte Handelspartner, die über eine Beziehung zu einem Endkunden verfügen, können diese Beziehung über das Partner Center entfernen.

Die Aktionen in den oben aufgeführten Szenarien müssen von einem Partner Center-Administrator-Agent ausgeführt werden.

### <a name="next-steps"></a>Nächste Schritte:

- Lesen Sie die Informationen in der Operations Readiness-Katalogsammlung: Aktualisierte Funktion zum Entfernen von Beziehungen im Partner Center.
- Leiten Sie diese Informationen an das geeignete Team in Ihrem Unternehmen weiter, damit die Prozesse überprüft und aktualisiert werden können.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>Das standardmäßige Fakturierungsintervall wird von der jährlichen in die monatliche Abrechnung geändert.

### <a name="categories"></a>Kategorien

- Datum: 17.06.2020
- Typ: Funktionen
- Status: Bald verfügbar

### <a name="summary"></a>Zusammenfassung

Das standardmäßige Fakturierungsintervall wird in der Partner Center-Einkaufsoberfläche von der jährlichen in die monatliche Abrechnung geändert.

### <a name="impacted-audience"></a>Zielgruppe

Direkte CSP-Anbieter (Cloud Solution Provider) und indirekte Anbieter

### <a name="details"></a>Details

Die Einstellung für das Fakturierungsintervall bestimmt, wie oft dem Partner die erworbenen Dienste in Rechnung gestellt werden. Partner sehen diese Einstellung im Schritt **Überprüfen**, wenn sie im Partner Center-Katalog Einkäufe tätigen. Diese Einstellung hat standardmäßig den Wert **Jährlich**, da die jährliche Abrechnung für lizenzbasierte Dienste eingeführt wurde. Das Partner Center-Team ändert diesen Standard ab dem 17. Juni aufgrund von Partnerfeedback von **Jährlich** in **Monatlich**, um die Erwartungen der Partner besser zu berücksichtigen. Partner können diese Einstellung auf der Seite „Überprüfen“ ändern, bevor sie eine Transaktion übermitteln, oder [auf der Seite „Abonnement verwalten“, nachdem eine Transaktion ausgeführt wurde]( https://docs.microsoft.com/partner-center/billing-annual-monthly#change-billing-frequency). Beachten Sie, dass Angebote, die nur die jährliche Abrechnung unterstützen, weiterhin standardmäßig jährlich abgerechnet werden. Diese geänderte Standardeinstellung gilt nur für die Benutzeroberfläche des Partner Center-Portals. Die Funktionsweise der Partner Center-APIs ändert sich nicht.

Wir hoffen, durch diese Änderung das Risiko zu verringern, dass Partner versehentlich jährlich abgerechnete Abonnements erwerben.

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>Änderungen an Microsoft 365 F1

### <a name="categories"></a>Kategorien

- Datum: 15.06.2020
- Typ: Angebote/Märkte
- Status: Jetzt verfügbar

### <a name="summary"></a>Zusammenfassung

Microsoft 365 F1 wird mit neuen Diensten aktualisiert. Am 15. Juni wird ein neues Angebot veröffentlicht, das die vorherige F1-Version ersetzt.

### <a name="impacted-audience"></a>Zielgruppe

Direkte CSP-Anbieter (Cloud Solution Provider) und indirekte Anbieter

### <a name="details"></a>Details

Microsoft 365 F1 ist seit April 2020 im Partner Center verfügbar. Es wurden Verbesserungen vorgenommen, indem neue Dienste unterstützt werden: Exchange-Kiosk- und MS Search-Funktionen. Die neuen Dienste werden automatisch für alle vorhandenen Abonnements basierend auf den ursprünglichen F1-Angeboten bis Ende Juni bereitgestellt. Die ursprünglichen F1-Angebote werden bei neu erworbenen Abonnements am 15. Juni durch die neuen F1-Angebote (unten) ersetzt.

Ursprüngliche F1-Angebote:

   |**Angebotsname**|**Angebots-ID**|**Materialkennung**|**Bereitstellungs-ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI-00005|M365_F1|
   |Microsoft 365 F1 – Testversion|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

Partner, die neue F1-Abonnements erwerben möchten, sollten ab dem 15. Juni die unten aufgeführten Angebots-IDs verwenden. Partner finden diese IDs im Partner Center-Katalog. Partner, die die Partner Center-APIs nutzen, können auch die unten aufgeführten Informationen verwenden.

Neue F1-Angebote:

   |**Angebotsname**|**Angebots-ID**|**Materialkennung**|**Bereitstellungs-ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|17b4a1d6-2c65-478e-a046-51cf40d25d76|1PI-00012|M365_F1_COM|
   |Microsoft 365 F1 – Testversion|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

### <a name="call-to-action"></a>Handlungsaufforderung
Partner mit dem aktuellen Microsoft 365 F1-Produkt müssen nichts unternehmen. Dienstpläne für Exchange-Kiosk und MS Search werden bis Ende Juni mit den vorhandenen Dienstplänen abgeglichen.

Partner, die Microsoft 365 F1 nach dem 15. Juni erwerben möchten, sollten anhand der Informationen zu den neuen F1-Angeboten sicherstellen, dass sie das richtige Angebot erwerben. Die ID für neue F1-Angebote und weitere Informationen werden am 1. Juli in der CSP-Preisliste für Juli aktualisiert.
