# <a name="use-partner-center-to-enroll-as-a-microsoft-learning-partner"></a>Registrieren als Microsoft Learning Partner mit Partner Center

**Geeignete Rollen:** MPN-Partneradministrator | Kontoadministrator

In diesem Artikel werden die verschiedenen Möglichkeiten beschrieben, wie sich ein Partner im Microsoft Partner Network für das Learning Partner-Programm registrieren kann.

Der Erfolg des Learning Partner Program wird anhand von Key Performance Indicators (KPIs) gemessen, die darauf ausgelegt sind, Learning-Partner zu beloben, wenn sie bestimmte Microsoft-Schulungen abschließen. KPIs werden basierend auf partnerbezogenen Käufen von qualifizierten Official Microsoft Learning Products (OMLP) aus dem Courseware Marketplace nachverfolgt.

Partner können Partner Center verwenden, um sich für das Learning Partner-Programm zu registrieren. Dies erfolgt über die Lernoption unter dem Microsoft Action Pack-Abonnement oder unter der Lernoption, die bestimmten Silver- oder Gold-Kompetenzen zugeordnet ist.

Dieser Prozess ist vollständig in Partner Center automatisiert und erfordert keine Geschäftsanwendung mehr, wie er in Partner Membership Center verwendet wurde.

Beachten Sie bei der Registrierung für das Learning Partner Program folgende Punkte:

- Die KPI-Anforderungen für das Learning Partner Program können jährlich geändert werden.

- Lern-KPIs müssen in Partner Center dem ersten Jahr der Registrierung erfüllt werden.

- **Ressourcen für neue Partner:** Wenn Sie ein neuer Partner sind, der daran interessiert ist, eine Kompetenz unter der Option Learning zu erwerben, können Sie mehr über den Prozess mit der Ressourcensammlung der [Learning-Option](https://partner.microsoft.com/asset/collection/learning-option-enrollment#/)erfahren.

Weitere Informationen finden Sie im [Partnerblogbeitrag](https://blogs.partner.microsoft.com/mpn/responding-to-covid-19-microsoft-partner-network/).

Programmdokumente und monatliche Aufzeichnungen von Communityaufrufen finden Sie auch auf unserer [Seite "Lernressourcen".](https://partner.microsoft.com/marketing/learning-resources#/)

## <a name="enroll-with-the-learning-action-pack"></a>Registrieren mit dem Learning-Action Pack

Learning Action Pack ist der Einstieg in das Learning Partner-Programm auf Basisebene. Wenn Sie noch nicht mit der Lernlinie arbeiten und Ihre Organisation noch keine Microsoft Courseware-Schulungen durchgeführt hat, bietet Microsoft Ihnen die Möglichkeit, sich als Microsoft Learning zu registrieren. Sie müssen lediglich die Learning-Action Pack.

Learning Action Pack unterstützt Partner, die Schulungsdienste, Angebote oder Rabatte auf dem Marketplace für Kursware anbieten möchten, sowie Vorteile, die Partnern helfen sollen, expertenübergreifend für verschiedene Microsoft-Produktlinien zu werden.

Nicht-Lernpartner können Kurssoftware nur für internes Training zum vollen Preis erwerben. Wenn Sie die Inhalte außerhalb Ihrer Organisation liefern möchten, benötigen Sie eine Microsoft Certified Trainer (eine MCT) und müssen sich für die Lernumgebung Action Pack.

Sobald Sie die erforderlichen KPIs erreicht haben, können Sie die zusätzlichen Anforderungen erfüllen und ein Upgrade auf eine Silver/Gold-Lernoption durchführen.

- Um sich für learning Action Pack in Partner Center qualifizieren zu können, müssen Sie über mindestens eine (1) MCT verfügen. Sie müssen keine anderen Lern-KPIs erfüllen.
- Um Ihre Learning-Action Pack aktivieren, führen Sie die folgenden Schritte aus:
   1. Führen Sie die Schritte unter [Kaufen oder Erneuern eines Microsoft Action Pack abonnements aus.](mpn-get-action-pack.md)
   2. Wenden Sie sich [an das Supportteam von Partner Frontline,](https://partner.microsoft.com/support) um Ein neues Kontoprofil zu erstellen. Dadurch erhalten Sie Zugriff auf das Microsoft Courseware-Training auf Courseware Marketplace.

## <a name="enroll-with-a-silver-competency-learning-option"></a>Lernoption "Mit Silber-Kompetenz registrieren"

Lernpartner, die eine Silver-Mitgliedschaftsstufe erreichen möchten, können sich bei der Silver Learning-Option registrieren, die einer bestimmten Kompetenz zugeordnet ist. Dazu müssen sie die Silver Learning-Optionsanforderungen für diese spezifische Kompetenz erfüllen.

> [!NOTE]
> Weitere Informationen dazu, welche Kompetenzen eine Silver- oder Gold Learning-Option bieten, finden Sie auf der neuesten [Seite zu Kompetenzen.](https://partner.microsoft.com/membership/competencies) 

Um sich für die Learning-Option einer bestimmten Silver-Kompetenz zu qualifizieren und sich für diese zu registrieren, überprüfen Sie die Silver-Anforderungen dieser Kompetenz, um sicherzustellen, dass Sie diese erfüllen. Dann führen Sie folgende Schritte aus:

- Halten Sie sich an **die Leistungsanforderungen,** die in den Leistungsrichtlinien für das Lernprogramm beschrieben sind. Diese Richtlinien sind auf der [Lernressourcen-Website](https://partner.microsoft.com/marketing/learning-resources#/)verfügbar.

- Erfüllen der **Microsoft Certified Trainer-Anforderungen (MCT)**

- Erfüllen der **Prüfungsanforderungen**

- Melden Sie sich beim Partner Center [Dashboard](https://partner.microsoft.com/dashboard)an, und wählen Sie dann im linken Navigationsmenü **MPN** und dann **Mitgliedschaftsangebote** aus. Führen Sie dann die [folgenden Schritte](mpn-get-action-pack.md) aus, um die Silver-Kompetenzstufe zu abonnieren. Dieser Artikel zur Zahlung von [Kompetenzgebühren](mpn-pay-fee-silver-gold-competency.md) kann ebenfalls nützlich sein.

## <a name="enroll-with-a-gold-competency-learning-option"></a>Registrieren mit einer Gold-KompetenzLernoption

Bevor Sie sich mit der Learning-Option "Gold-Kompetenz" registrieren können, müssen Sie sicherstellen, dass Sie die Anforderungen dieser spezifischen Lernoption auf Goldebene erfüllen.

Die Registrierung bei einer Learning-Option für Gold-Kompetenz (oder deren Verlängerung) hat andere Anforderungen als die Registrierung bei der Learning Action Pack- oder Silver-Kompetenzlernoption. Um die Anforderungen für eine Gold-Kompetenz zu erfüllen, müssen Sie auch die Lern-KPI-Anforderungen dieser Kompetenz auf Goldebene erfüllen. Wenn Sie die KPIs oder andere Anforderungen nicht erfüllt haben, müssen Sie sich für die Option Learning Action Pack oder Silver registrieren, je nachdem, welche Anforderungen erfüllt sind, wenn Sie Lernpartner werden möchten.

> [!NOTE]
> Möglicherweise sind Sie bereits in einer neuen Kompetenz registriert und haben bereits die Silver Learning-Optionenanforderungen für diese Kompetenz erfüllt. Wenn Sie dann die Lernoption auf Goldebene für diese Kompetenz erreichen möchten, müssen Sie die KPIs auf Gold-Ebene und andere Gold-Level-Anforderungen dieser Kompetenz erfüllen. Sobald die Gold-Level-Anforderungen erfüllt sind, können Sie möglicherweise den MPN-Angebot – Upgrade von [Silver-to-Gold-Rabatt](mpn-pay-fee-silver-gold-competency.md#apply-upgrade-discount-when-moving-from-silver-to-gold)nutzen.

## <a name="frequently-asked-questions-about-learning-partner-enrollment"></a>Häufig gestellte Fragen zur Registrierung von Learning-Partnern

### <a name="q-do-i-need-to-submit-a-business-application-plan-in-partner-center-to-enroll-in-a-learning-option"></a>Q. Muss ich einen Geschäftsanwendungsplan in Partner Center übermitteln, um mich für eine Learning-Option zu registrieren?

Nein. In Partner Center ist ein Geschäftsanwendungsplan nicht erforderlich, um eine Kompetenz über die Lernoption zu erhalten. Sie müssen die Lern-KPIs aus dem ersten Jahr der Registrierung erfüllen.

### <a name="q-if-i-am-a-net-new-partner-and-decide-to-buy-courseware-trainings-or-certifications-before-applying-for-the-learning-option-will-the-kpis-be-recognized-by-partner-center-automatically"></a>F: Wenn ich ein net-new-Partner bin und mich entschließe, Kurssoftware, Schulungen oder Zertifizierungen zu erwerben, bevor ich mich für die Lernoption bewirb, werden die KPIs von Partner Center erkannt?

Ja, die KPIs sollten in der -Partner Center.

### <a name="q-where-should-i-go-in-case-there-is-a-discrepancy-between-kpis-in-partner-center-under-the-learning-option-and-my-kpi-dashboard"></a>Q. Wohin sollte ich gehen, falls es zu einer Abweichung zwischen KPIs in Partner Center Lernoption und meinem KPI-Dashboard kommt?

Wenden Sie [sich an den Partner Frontline-Support.](https://partner.microsoft.com/support) Sie müssen die folgenden Details (aus Ihrem KPI-Dashboard) bereitstellen, damit wir Ihre KPIs nachverfolgen können:

- Month (Monat)
- Jahr
- OMLP-Typ
- Bestelleinheit
- Rechnungsnr.
- KPI-Wert
- Umsatzwert

KPI-Berichte werden in der Mitte des Monats einmal im Monat aktualisiert. Die Berechtigung zur Erneuerung des Lernprogramms wird basierend auf Ihrem Jahrestag für KPIs gemessen, die in den letzten 12 Monaten der Leistungsaktivität erzielt wurden. Beispielsweise werden ab dem 15. Januar 2020 KPI-Daten für den Zeitraum von Januar 2019 bis Dezember 2019 angezeigt. Wenn Ihre Rechnung vom Januar 2020 stammt, werden die entsprechenden KPIs ab dem 15. Februar 2019 angezeigt. **Bestellbestätigungen und Partnerberichte werden nicht akzeptiert.** Nur tatsächliche Rechnungen werden akzeptiert.

### <a name="q-does-a-one-day-training-equate-to-one-kpi"></a>Q. Entspricht ein eintägiges Training einem KPI?

Die Erfüllung eines KPI bezieht sich nicht auf die Dauer des Trainings. Weitere Informationen zur Messung von KPIs finden Sie in den Leistungsbeispielen im [Lernpartner-Registrierungsleitfaden.](https://partner.microsoft.com/asset/collection/learning-option-enrollment#/)

### <a name="q-why-dont-i-appear-as-a-microsoft-learning-partner-in-the-learning-partner-directory"></a>Q. Warum wird ich nicht als Microsoft Learning Partner im Lernpartnerverzeichnis angezeigt?

Möglicherweise müssen Wir Sie dem [Lernpartnerverzeichnis hinzufügen.](/learn/certifications/partners) Sie können dann auf der Verzeichnisseite des Lernpartners einen Link zu Ihrer Website erstellen.

Dies ist ein manueller Prozess, der nur einmal pro Monat in der dritten Woche übermittelt wird. Das nachfolgende Update wird am Ende der ersten Woche des folgemonats angezeigt. Wenden Sie sich an den [Frontline-Support](https://partner.microsoft.com/support)des Partners, um Dies zu beheben. Bereiten Sie sich darauf vor, die folgenden Details anzugeben:

- Der Name, der im Verzeichnis angezeigt werden soll.

- Das zugeordnete Land

- Die URL für die zugeordnete Landing Page mit Microsoft-Themen

   > [!NOTE]
   > Die URL sollte zu einer partnereigenen Website mit einer Microsoft-Landing Page lauten. Auf der Microsoft-spezifischen Seite muss außerdem Folgendes angezeigt werden:
   > - Nur **"Partner X"**
   > - **Keine Mitbewerber anzeigen**
   > - Im **Besitz von** "Partner X"
