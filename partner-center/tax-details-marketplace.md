# <a name="tax-details-for-commercial-marketplace-publishers"></a>Steuerliche Aspekte für kommerzielle Marketplace-Herausgeber

**Geeignete Rollen**

- Globaler Administrator

Auf dieser Seite werden die Steuerzuständigkeiten und Steueraspekte für Herausgeber (Herausgeber) des kommerziellen Marketplace und Consumer Marketplace-Entwickler (Entwickler) im Zusammenhang mit Endkundenkäufen von Herausgeber-/Entwicklerprodukten im Rahmen der [Microsoft Azure Marketplace Herausgebervereinbarung](https://go.microsoft.com/fwlink/p/?LinkID=699560) bzw. der [Vereinbarung für App-Entwickler](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE4o4bH)erläutert.  Die folgenden Informationen dienen zusätzlich zu den Geschäftsbedingungen in der Microsoft-Herausgebervereinbarung und den Vereinbarung für App-Entwickler und sollen diese nicht ersetzen.  Sofern nicht anders angegeben, gelten die unten angegebenen Informationen sowohl für Herausgeber im kommerziellen Marketplace als auch für Consumer Marketplace-Entwickler, die ihre Produkte/Angebote über Microsoft Marketplaces zum Verkauf zur Verfügung stellen. 

## <a name="end-customer-taxation"></a>End Customer Auserlangung

Die Zuständigkeiten von Microsoft und Herausgebern/Entwicklern für Vertrieb, Nutzung, Waren und Dienstleistungen, Mehrwertsteuer oder ähnliche Steuern (sofern vorhanden) für Endkundenverkäufe **(Endkundenabsicht)** hängen von dem Land ab, in dem Herausgeber/Entwickler Produkte/Angebote für Endkunden über die Microsoft Marketplace zur Verfügung stellen. In der folgenden Tabelle sind die Umstände und zuständigkeiten für die Endkundenausbindung in Ländern aufgeführt, deren Verkäufe über Microsoft Marketplace berechtigt sind.  Die folgenden Informationen können geändert werden, und Herausgeber/Entwickler sind dafür verantwortlich, diese Seite auf aktuelle, genaue Informationen zu überprüfen.

In ländern, die von **Herausgebern/Entwicklern verwaltet werden,** fungiert Microsoft als Agent oder Provisionagent (sofern zutreffend) des Herausgebers/Entwicklers. Herausgeber/Entwickler bestätigen und erklären sich damit einverstanden, dass Herausgeber/Entwickler die alleinige Verantwortung dafür tragen, die erforderliche Konformität für und die Verwaltung der Endkunden zu bestimmen, einschließlich Registrierung, Steuerberechnung, Steuererfassung, Steuerausstellung, Überprüfung des Geschäftsstatus von Kunden und Bereitstellen von Steuerrechnungen für Kunden. Herausgeber/Entwickler bestätigen, dass Microsoft in Bezug auf jeden Verkauf in einem von Herausgebern/Entwicklern verwalteten Land derzeit möglicherweise nicht in der Lage ist, die Transaktionsdetails zur Verfügung zu stellen, die Herausgeber/Entwicklern zum Bereitstellen von Steuerrechnungen für Kunden erforderlich sind. 

In **von Microsoft verwalteten Ländern** fungiert Microsoft als Agent des Herausgebers/Entwicklers. Microsoft übernimmt die Verantwortung für die Verwaltung der Endkundenesenkung, einschließlich der Berechnung, Einsammlung und/oder Steuerersteuerung. Für den kommerziellen Marketplace kann dies auch die Validierung des Geschäftsstatus von Endkunden umfassen, indem Steuerregistrierungsnummern oder Freistellungszertifikate erhalten werden. Microsoft verwaltet die Endkundenbindung entweder über lokale Partner oder remote verbundene Unternehmen, die lokal registriert sind. Microsoft-Rechnungen unter der Registrierungsnummer von Microsoft. Microsoft schließt alle Steuern ein, die von Endkunden auf der Steuererklärung von Microsoft einbeziehen werden.

Microsoft übernimmt keine Garantie dafür, dass die Aktionen von Microsoft die Herausgeber-/Entwicklerverpflichtungen in von Microsoft verwalteten Ländern vollständig erfüllen.  Für alle von Microsoft verwalteten Länder empfiehlt Microsoft Herausgebern/Entwicklern dringend, mit ihren eigenen Beratern zusammen zu arbeiten, um zu ermitteln, ob die von Microsoft verwaltete Steuererklärung die Konformitätsanforderungen von Herausgebern/Entwicklern ausreichend erfüllt. Dies ist besonders wichtig für alle von Microsoft verwalteten Länder, in denen Herausgeber/Entwickler ihre Produkte/Angebote verkaufen.  Microsoft fungiert als Agent des Herausgebers/Entwicklers in von Microsoft verwalteten Ländern.

In **Handelspartnern** fungiert Microsoft als Vertriebspartner und nicht als Agent des Herausgebers/Entwicklers, wenn es darum geht, Herausgeber-/Entwicklerprodukte/-angebote über die Microsoft Marketplace für Endkunden verfügbar zu machen.  Herausgeber/Entwickler sind für den Verkauf von Produkten/Angeboten an Microsoft, für Waren und Dienstleistungen, für die Wertschöpfung oder ähnliche Steuern (sofern vorhanden) verantwortlich, und Microsoft ist für alle Endkundengebühren für den Weiterverkauf an Endkunden verantwortlich.

## <a name="tax-considerations-on-payouts"></a>Steueraspekte bei Auszahlungen

In ländern, die von Herausgebern/Entwicklern verwaltet werden, ist die Microsoft-Auszahlung an Herausgeber/Entwickler ohne die Store-Gebühr und alle anwendbaren Steuern, die folgende Steuern umfassen können: Steuereinbehaltung und Verkäufe, Nutzung, Waren und Dienstleistungen, Mehrwert- oder ähnliche Steuern auf die Ladengebühr.

In den von Microsoft verwalteten Ländern und Wiederverkäufern ist die Microsoft-Auszahlung an Herausgeber/Entwickler ohne die Ladengebühr und alle anwendbaren Steuern, die eine der folgenden Enthalten können: Endkundensteuer, Steuereinbehaltung und Verkäufe, Nutzung, Waren und Dienstleistungen, Mehrwert- oder ähnliche Steuern auf die Ladengebühr. In [den Berichten,](payout-statement.md) die Microsoft Herausgebern/Entwicklern zur Verfügung stellt, wird der Betrag dieser Steuern identifiziert. 

Microsoft sammelt Steuern auf Ladengebühren in den folgenden Ländern, in denen Herausgeber/Entwickler Produkte/Angebote für Endkunden über die folgenden Microsoft Marketplace zur Verfügung stellen:

|----------------|-----------------------------|-----------------------|
