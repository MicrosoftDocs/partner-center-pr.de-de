# <a name="tax-details-for-commercial-marketplace-publishers"></a>Steuerliche Aspekte für kommerzielle Marketplace-Herausgeber

**Geeignete Rollen**: Globaler Administrator

Auf dieser Seite werden die Steuerzuständigkeiten und Steueraspekte für Herausgeber des kommerziellen Marketplace (Herausgeber) und Consumer Marketplace-Entwickler (Entwickler) im Zusammenhang mit Endbenutzerkäufen von Herausgeber-/Entwicklerprodukten im [Microsoft Azure Marketplace Herausgebervertrag](/legal/marketplace/msft-publisher-agreement) bzw. im [Vereinbarung für App-Entwickler](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE4o4bH)erläutert.  Die folgenden Informationen dienen zusätzlich zu den Geschäftsbedingungen in der Microsoft-Herausgebervereinbarung und den Vereinbarung für App-Entwickler und sollen diese nicht ersetzen.  Sofern nicht anders angegeben, gelten die folgenden Informationen sowohl für Herausgeber im kommerziellen Marketplace als auch für Consumer Marketplace-Entwickler, die ihre Produkte/Angebote über Microsoft Marketplaces zum Verkauf zur Verfügung stellen. 

## <a name="end-customer-taxation"></a>End Customer Auserlangung

Die Zuständigkeiten von Microsoft und Herausgebern/Entwicklern für Vertrieb, Nutzung, Waren und Dienstleistungen, Mehrwertsteuer oder ähnliche Steuern( falls vorhanden) für Endkundenverkäufe **(Endkundenzuständigkeit)** hängen von dem Land ab, in dem Herausgeber/Entwickler Produkte/Angebote für Endkunden über die Microsoft Marketplace zur Verfügung stellen. In der folgenden Tabelle sind die Umstände und zuständigkeiten für die Endkundenausbindung in Ländern aufgeführt, deren Verkäufe über Microsoft Marketplace berechtigt sind.  Die folgenden Informationen können geändert werden, und Herausgeber/Entwickler sind dafür verantwortlich, diese Seite auf aktuelle, genaue Informationen zu überprüfen.

In ländern, die von **Herausgebern/Entwicklern verwaltet werden,** fungiert Microsoft als Agent oder Provisionagent (sofern zutreffend) des Herausgebers/Entwicklers. Herausgeber/Entwickler bestätigen und stimmen zu, dass Herausgeber/Entwickler die alleinige Verantwortung für die Bestimmung der für und die Verwaltung von End Customer Publishers erforderlichen Compliance haben, einschließlich, aber nicht beschränkt auf Registrierung, Steuerberechnung, Steuereinbehaltung, Steuerzuteilung, Überprüfung des Geschäftsstatus von Kunden und Bereitstellung von Steuerrechnungen für Kunden. Herausgeber/Entwickler bestätigen, dass Microsoft in Bezug auf jeden Verkauf in einem vom Herausgeber/Entwickler verwalteten Land derzeit möglicherweise nicht in der Lage ist, die Transaktionsdetails anzugeben, die Herausgeber/Entwickler benötigen, um Kunden Steuerrechnungen bereitzustellen. 

In **von Microsoft verwalteten Ländern** fungiert Microsoft als Agent des Herausgebers/Entwicklers. Microsoft übernimmt die Verantwortung für die Verwaltung der Endkundenzuteilung, einschließlich der Berechnung, Einsammlung und/oder Ausgabe bestimmter Steuern. Für den kommerziellen Marketplace kann dies auch die Überprüfung des Geschäftsstatus von Endkunden umfassen, indem Steuerregistrierungsnummern oder Steuerausnahmezertifikate erhalten werden. Microsoft verwaltet die Endkundenbindung entweder über lokale Niederlassungen oder Remoteunternehmen, die lokal registriert sind. Microsoft-Rechnungen unter der Registrierungsnummer von Microsoft. Microsoft schließt alle Steuern ein, die von Endkunden auf die Steuererklärung von Microsoft erhoben werden.

Microsoft übernimmt keine Garantie dafür, dass die Aktionen von Microsoft die Herausgeber-/Entwicklerverpflichtungen in von Microsoft verwalteten Ländern vollständig erfüllen.  Für alle von Microsoft verwalteten Länder empfiehlt Microsoft dringend, dass Herausgeber/Entwickler mit ihren eigenen Steuerberatern zusammenarbeiten, um zu ermitteln, ob die von Microsoft verwaltete Steuerzustellung die Complianceanforderungen von Herausgebern/Entwicklern ausreichend erfüllt. Dies ist besonders wichtig für alle von Microsoft verwalteten Länder, in denen Herausgeber/Entwickler ihre Produkte/Angebote verkaufen.  Microsoft fungiert als Agent des Herausgebers/Entwicklers in von Microsoft verwalteten Ländern.

In **Handelspartnern** fungiert Microsoft als Vertriebspartner und nicht als Agent des Herausgebers/Entwicklers, wenn es darum geht, Herausgeber-/Entwicklerprodukte/-angebote über die Microsoft Marketplace für Endkunden verfügbar zu machen.  Herausgeber/Entwickler sind für den Verkauf, die Nutzung, waren und Dienstleistungen, die Wertschöpfung oder ähnliche Steuern (sofern vorhanden) für den Verkauf von Produkten/Angeboten an Microsoft verantwortlich, und Microsoft ist für alle Endkundengebühren für den Weiterverkauf an Endkunden verantwortlich.

## <a name="tax-considerations-on-payouts"></a>Steueraspekte bei Auszahlungen

In ländern, die von Herausgebern/Entwicklern verwaltet werden, ist die Microsoft-Auszahlung an Herausgeber/Entwickler ohne die Store-Gebühr und alle anwendbaren Steuern, die folgende Steuern umfassen können: Steuereinbehaltung und Verkäufe, Nutzung, Waren und Dienstleistungen, Mehrwert- oder ähnliche Steuern auf die Ladengebühr.

In den von Microsoft verwalteten Ländern und Wiederverkäufern ist die Microsoft-Auszahlung an Herausgeber/Entwickler ohne die Ladengebühr und alle anwendbaren Steuern, die eine der folgenden Enthalten können: Endkundensteuer, Steuereinbehaltung und Verkäufe, Nutzung, Waren und Dienstleistungen, Mehrwert- oder ähnliche Steuern auf die Ladengebühr. In [den Berichten,](payout-statement.md) die Microsoft Herausgebern/Entwicklern zur Verfügung stellt, wird der Betrag dieser Steuern identifiziert. 

Microsoft sammelt Steuern auf Ladengebühren in den folgenden Ländern, in denen Herausgeber/Entwickler Produkte/Angebote für Endkunden über die folgenden Microsoft Marketplace zur Verfügung stellen:

|**Country**|**Kommerzieller Marketplace**|**Consumer Marketplace**|
|----------------|-----------------------------|-----------------------|
|Australien|Steuern der Dienstgebühr|Steuern der Dienstgebühr|
|Canada|Steuern der Dienstgebühr|Steuern der Dienstgebühr|
|Mexiko|–|Steuer auf Dienstgebühr|
|Neuseeland|Steuern der Dienstgebühr|Steuern der Dienstgebühr|
