# <a name="value-added-tax-vat-information"></a>Informationen zur Umsatzsteuer (Value Added Tax, USt)

**Geeignete Rollen:** Kontoadministrator


Wenn Sie während des Registrierungsprozesses eine USt-IdNr. Partner Center müssen, finden Sie hier einige Informationen, um Ihnen den Einstieg zu ermöglichen.

## <a name="understanding-vat-numbers"></a>Grundlagen der Steuernummern

Eine Umsatzsteuernummer ist eine ID, die für Länder oder Regionen in der EU verwendet wird Weitere Informationen finden Sie auf der offiziellen WEBSITE der [Europäischen](http://ec.europa.eu/taxation_customs/vies/vieshome.do)Union .

## <a name="location-specific-vat-information"></a>Ortsspezifische USt-Informationen

Die folgenden Details gelten nur für die unten aufgeführten Länder oder Regionen.

### <a name="european-union"></a>Europäische Union

In den Mitgliedstaaten der Europäischen Union müssen Partner ihre gültige USt-IdNr. angeben, um eine ordnungsgemäße Steuerschuldumkehr und Abrechnung zu gewährleisten. Die USt-IdNr. muss nachweislich gültig sein.

Microsoft unterstützt keine Gutschrift und keine Neurechnung. Bevor Sie Abonnements verkaufen, müssen Sie Ihre [USt-IdNr.](organization-tax-info.md) in Partner Center.

### <a name="south-africa"></a>Südafrika

Microsoft wendet jetzt die Standardmäßige Umsatzsteuer auf digitale Inhalte und Dienstleistungen an, die von Partnern in Südafrika erworben wurden. Um die Umsatzsteuer von Ihren Rechnungen auszuschließen und diese selbst zu verwalten, müssen Sie Ihr Partnerabrechnungsprofil aktualisieren, indem Sie Microsoft Ihre [USt-IdNr. senden.](organization-tax-info.md)

### <a name="turkey"></a>Türkei

Microsoft wendet jetzt die Standardmäßige Umsatzsteuer auf digitale Inhalte und Dienstleistungen an, die von Partnern in Europa erworben wurden. Um die Umsatzsteuer von Ihren Rechnungen auszuschließen und diese selbst zu verwalten, müssen Sie Ihr Partnerabrechnungsprofil aktualisieren, indem Sie Microsoft Ihre [USt-IdNr. senden.](organization-tax-info.md)
